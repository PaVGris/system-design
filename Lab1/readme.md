## Вариант 2 
Магазин https://ozon.ru/

Приложение должно содержать следующие данные:
- Пользователь
- Товар
- Корзина

Реализовать API:
- Создание нового пользователя
- Поиск пользователя по логину
- Поиск пользователя по маске имя и фамилии
- Создание товара
- Получение списка товаров
- Добавление товара в корзину
- Получение корзины для пользователя

## 1. Роли пользователей
-   **Client (Покупатель)** — регистрируется, ищет товары, формирует корзину, смотрит в каталог.
-   **Seller (Продавец)** — добавляет новые товары в каталог, ищет клиентов.

## 3. Список контейнеров
-   **Web Browser** - веб-интерфейс, через который пользователи взаимодействуют с системой.
-   **API Gateway** - шлюз на базе Userver Framework. Маршрутизирует запросы к сервисам.
-   **User Service** - сервис (Userver) для управления учетными записями и аутентификации.
-   **Item Service** - сервис (Userver) для управления каталогом товаров.
-   **Cart Service** - сервис (Userver) для управления состоянием корзин.
-   **PostgreSQL** - основная БД для хранения профилей пользователей и данных о товарах.
-   **Redis** — in-memory БД для быстрого доступа к активным корзинам.
